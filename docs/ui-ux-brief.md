# UI/UX Brief

## Brand vibe
Three adjectives: clean, reliable, efficient
Inspiration apps: Facebook login simplicity, lightweight CRM dashboard, mobile-first operations panel

## Colour tokens
primary: #1877F2
secondary: #0F172A
accent: #E8F1FF
success: #16A34A
warn: #D97706
error: #DC2626
bg: #F5F7FB
text: #111827
muted: #6B7280

## Type scale
Font family: Poppins
h1: 32px / 700
h2: 24px / 600
body: 16px / 400
small: 14px / 500

## Spacing & radius
Spacing unit: 4px
Radius: sm 4px, md 8px, lg 16px, pill 9999px

## Components
Required: Button, Input, Card, Table, Dialog, Toast, EmptyState, Skeleton, ErrorBoundary

## Voice & tone
Microcopy style: clear and operational (example: "Lead assigned.")
Error tone: calm and direct (example: "Upload failed. Try again.")

## Accessibility floor
- Contrast: 4.5:1 minimum for body text
- All interactive elements reachable via keyboard
- Focus rings always visible
- Forms have aria-describedby for errors
- Touch targets at least 44px high on mobile
